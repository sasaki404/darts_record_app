# ダーツ記録アプリ

自宅やゲームセンター等でダーツをする際に、プレイを記録するためのアプリです。カウントアップ、01、クリケットなどの基本的なゲームを含む5種類のモード対応する予定です。

## 主な特徴

### オフライン対戦

端末1つで複数人と対戦をすることができます。
